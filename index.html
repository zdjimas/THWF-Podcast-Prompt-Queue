Prompt Queue — Zero Cost

This is a single-page static app for GitHub Pages. It uses one admin code to unlock both submission and admin controls, stores the queue locally in the browser, and can export or import an encrypted JSON backup file.

No backend • No Fly.io • No monthly cost

Unlock access

Enter the shared admin code. The app compares a SHA-256 hash in the browser. This is acceptable for a lightweight private tool, but it is not server-grade security.

To change the admin code later, update ADMIN_CODE_HASH in the script with a new SHA-256 hash.

Submit prompt

New submissions are automatically marked pending.

Queue controls

Export gives you a real encrypted file backup. Import restores it. This is the closest free static-file approach to “saving to a file” without paying for a backend.
